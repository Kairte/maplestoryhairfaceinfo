통계 API와 연결된 운영용 대시보드입니다. 현재 선택한 탭 기준으로 최신 집계 결과와 연동 상태를 함께 확인할 수 있습니다.

최근 동기화
연결 확인 중 현재 선택한 통계 탭 기준으로 최신 집계를 확인합니다.
연동 상태 API 응답 대기 중 초기 로드 후 실제 집계 수치로 갱신됩니다.

운영 메모 영역 견본

오답률 보정 후보 이미지 유사도가 높은 항목, 색놀이 파생형, 동일 명칭 변형 아이템을 별도 태그로 관리할 수 있습니다.
어워즈 검수 메모 연도별 뷰티어워즈 인기 흐름과 재등장 빈도를 함께 적어두는 공간으로 활용 가능합니다.
향후 연결 지점 `/api/stats/rankings`, `/api/stats/items/:itemKey` 같은 외부 통계 API를 여기에 연결하는 구조를 가정했습니다.

현재 파일은 견본 UI입니다. 실제 운영 시에는 관리자만 수정할 수 있는 별도 저장 구조를 두고, 리스트·툴팁·랭킹 수치는 API에서 불러오도록 바꾸면 됩니다.
